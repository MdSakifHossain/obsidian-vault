# Ubuntu Look Customization

⬅️ [Ubuntu](./Ubuntu.md)

## Understanding

> Shell theme = system UI\
> GTK theme = app UI
>
> Outside apps → Shell Theme\
> Inside apps → GTK Theme

[Reference Video](https://www.youtube.com/watch?v=viffvWtMTdo)

## The Actual Part

- Install Extentions from `Extention Manager`
  - [User Themes by fmuellner](https://extensions.gnome.org/extension/19/user-themes/) **Most Important**
  - [Hide Top Bar by tuxor1337](https://extensions.gnome.org/extension/545/hide-top-bar/)
  - [Blur my Shell by aunetx](https://extensions.gnome.org/extension/3193/blur-my-shell/)
  - [Burn My Windows by Simon Schneegans](https://extensions.gnome.org/extension/4679/burn-my-windows/)
  - [Compiz alike magic lamp effect by hermes83](https://extensions.gnome.org/extension/3740/compiz-alike-magic-lamp-effect/)
  - [Compiz windows effect by hermes83](https://extensions.gnome.org/extension/3210/compiz-windows-effect/)
- Download Cursor Skin:
  - [ArcAurora Cursors](https://www.gnome-look.org/p/1665694)
- Download Shell Theme:
  - [Graphite-gtk-theme](https://github.com/vinceliuice/Graphite-gtk-theme)
- Download GTK Themes:
  - GTK theme will be the same as Shell Theme
- Download Icon Theme:
  - [McMojave-circle](https://www.gnome-look.org/p/1305429)
- Download Fonts (Inter, Poppins, DM Mono, Space Mono etc.) `Nerd Fonts`
  - Search: `how to install a font in ubuntu if i have several ttf files`
- Change Terminal Looks
